# Cobra NATS Projekt – Gesamtkontext

**Projekt**: Moderne, professionelle Web-UI für NATS + JetStream (v0.2.0)

**Tech-Stack**:
- Next.js 16 (App Router) + React 19
- TypeScript 5
- Tailwind CSS v4 + shadcn/ui (New York Style)
- NATS Client: `nats` v2.29.3 über Server Actions
- Zustand + TanStack Table + React Hook Form + Zod
- Playwright E2E Tests

**Wichtige Features**:
- Multi-Connection Management (none / user_pass / token)
- JetStream Streams + Consumer Management
- Key-Value Stores (KV)
- Object Stores (OS) mit Upload/Download
- Publish + Request-Reply
- Live Subject Monitor (SSE)

**Ziel**: Schnellste und schönste NATS/JetStream Verwaltungsoberfläche.
